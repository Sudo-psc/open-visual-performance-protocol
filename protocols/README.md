# OVPP Protocols

This directory contains the protocol in two languages:

- [English](open-visual-performance-protocol.md)
- [Portugues (PT-BR)](protocolo-aberto-desempenho-visual.md)

Both are Version 2.2 companion-product releases for *O Custo Invisível do Olho
Seco*, structurally equivalent section-by-section.

Use the version that fits your organization, then localize clinical thresholds,
legal language and referral pathways to your jurisdiction. Preserve the core
sequence: baseline, risk map, environment, pauses, referral, re-measurement.
